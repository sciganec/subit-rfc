# SUBIT Archetype Guide (Style A)  
*Canonical 64 Structural States*  
Version 2.0

This document defines the **64 canonical structural archetypes** used in SUBIT‑based systems.  
These archetypes are **not part of SUBIT RFC 2.0**.  
They represent a **reference implementation** of the 64 possible 6‑bit SUBIT states.

Each archetype includes:

- ID (0–63)  
- 6‑bit binary code  
- Family (N, F, S, T, M, C, V, D)  
- Style‑A name (single‑word, structural, technical)  
- Structural description (1–2 sentences)

Archetypes are **structural states**, not symbols, metaphors, or psychological types.
