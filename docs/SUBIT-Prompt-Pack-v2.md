# SUBIT Prompt Pack v2.0  
*Standardized Prompt Library for LLMs*

This document defines the **official prompt set** for activating and using the SUBIT Structural Bit Protocol inside any LLM or agentic system.

SUBIT is a six‑axis operator:

**Tension → Vector → Interaction → Meaning → Action → Structure**

All prompts in this pack enforce or utilize this structure.
