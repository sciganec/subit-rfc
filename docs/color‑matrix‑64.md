# SUBIT 64‑Color Matrix  
*Multiply‑Blend Matrix of Canonical Trigram Colors*  

This document defines the **64‑color SUBIT matrix**, generated by:

- the **8 canonical trigram colors** from SUBIT Color Standard v1.1  
- the **canonical trigram order** from SUBIT Isomorphism Standard v1.0  
- the **multiply‑blend rule**  

Each cell corresponds to a 6‑bit SUBIT state:

```
upper_trigram × lower_trigram
```

Colors are shown as **hex values**.

# 2. Multiply‑Blend Rule

For two colors:

```
C_upper = (Ru, Gu, Bu)
C_lower = (Rl, Gl, Bl)
```

The blended color is:

```
R = (Ru * Rl) / 255
G = (Gu * Gl) / 255
B = (Bu * Bl) / 255
```

Rounded to nearest integer.
